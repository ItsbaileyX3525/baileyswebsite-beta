Platformer 1.0

Platformer 1.0, mine and RangerRhino23's first game using the taptapir engine

Platformer 1.0 was made using the taptapir engine built by poke so that you could make ursina games on the mobile devices (as at the time ursina didn't have support for building to apk) and me and Ryan saw this as a chance to create a fun mobile and desktop game using a brand new engine on the block! Due to the fact that the engine had literally just came out there was limited features so we had to create collision detection and all of that by ourselves which was very hard considering we had only just started our coding journey like a year prior and we weren't very smart haha. But using a very jank system we created all of the collison and needed items! I really recommend you try the game out and see if its good! Originallly you would've wanted to change your refresh rate to 60hz otherwise the game would run weirdly (we didnt know about delta time and such at that point.) I have tried my best to fix it and it should run well no matter your refresh rate/frame rate!
